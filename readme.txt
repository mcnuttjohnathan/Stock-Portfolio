Program can be started by typing "python stock_portfolio.py" in the current directory

Summary: allows the user to buy and sell stocks using NASDAQ symbols. Owned stocks
are kept in a portfolio database, which can be view using the "stock portfolio"
option on the main menu, this also calculates whether the portfolio has increased
in value. The program keep a database with all the transaction that the user has 
preformed by selecting "transaction log". Lastly the program keep track of the 
trends for each symbol which is updated when the user checks their portfolio.

Example database included under username "example"
